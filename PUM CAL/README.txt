HOW TO USE.
1. First enter the syllabus name and code. Make sure to enter correctly as you cannot change later.
2. Enter the grade boundaries from the grade threshold sequentially. Note ** this program only supports till P5.
3. Enter the raw marks of the student. If it's AS Level and all papers are not complete, enter 0 for P4 and P5 ( the 4th and 5th input )
4. Enter the weighting factors for the papers. If there is no weighting factor for the time being, enter 1.


--THE PROGRAM THEN SPITS OUT A RESULTS.TXT FILE WITH THE COMPONENT NAME, CODE, GRADE AND PERCENTAGE UNIFORM MARK--
